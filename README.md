# OweHost

OweHost is a modern hosting control platform built with a
filesystem-first architecture and idempotent system design.

## Features
- Account & domain management
- Web server & PHP runtime control
- SSL, DNS, backups
- Filesystem-based recovery
- Event-driven auditing

## Tech Stack
- Backend: Go
- Frontend: React + TypeScript
- Web: Nginx, PHP-FPM
